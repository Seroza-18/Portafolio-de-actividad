555

Mario E. Cuan Rivera y Sebastián Rodríguez Zapata
Ing. Mecatrónica
Fecha: 15/ago/24
Arduino Electrónica

- Resumen -

En este proyecto de 555, integramos un temporizador 555 con un Arduino Uno para controlar el parpadeo de un LED. El objetivo principal fue utilizar el 555 en modo astable para generar una señal de pulso que controla el encendido y apagado del LED a intervalos regulares..

- Introducción -

En este proyecto práctico se utilizó el temporizador 555 para generar una señal de control que permite el encendido y apagado de un LED. El temporizador 555 es un circuito integrado ampliamente utilizado en diversas aplicaciones de temporización y generación de pulsos. El objetivo de esta práctica fue diseñar, construir y probar un circuito utilizando el 555 en modo astable, controlado por componentes externos como resistencias, capacitores y un potenciómetro.

- Materiales -

-Arduino Uno

-Temporizador 555

-Resistencias

-Condensadores

-LED

-Protoboard y cables de conexión

- Desarrollo -

El proyecto se centró en el diseño de un circuito basado en el temporizador 555, que genera una señal de onda cuadrada para controlar el parpadeo de un LED. Se utilizaron componentes pasivos, como resistencias y capacitores, para ajustar la frecuencia del parpadeo. Además, se incluyó un potenciómetro para variar manualmente esta frecuencia, proporcionando un control dinámico sobre la velocidad del parpadeo del LED.

Simulación

Diseño

El diseño del circuito incluyó los siguientes componentes:

-555 IC en configuración astable para generar la señal de salida.

-LED para visualizar la señal generada.

-Potenciómetro para ajustar la frecuencia.

-Resistencias y capacitores para definir los tiempos de carga y descarga, que determinan la frecuencia de la señal.

Construcción

En la construcción se montó el circuito en una protoboard. Las imágenes adjuntas muestran el montaje del temporizador 555 junto con el potenciómetro, resistencias, capacitores y el LED.

-En la primera imagen se puede observar el cableado detallado para la correcta configuración del temporizador en modo astable.

-La segunda imagen muestra una versión más simplificada del montaje, donde se incluye el potenciómetro para el ajuste de frecuencia.

- Resultados -

El circuito funcionó como se esperaba. El temporizador 555, en su configuración astable, generó una señal de onda cuadrada que permitió que el LED parpadeara a una frecuencia controlada. Al ajustar el potenciómetro, se pudo observar cómo la frecuencia de parpadeo aumentaba o disminuía, lo que demostró el correcto funcionamiento del temporizador y la sensibilidad del circuito a los valores de resistencia y capacitancia.

- Conclusiones -

El temporizador 555 es una herramienta eficiente y versátil para generar señales de control en proyectos de electrónica. En este proyecto, se comprobó la facilidad con la que se puede utilizar para controlar el parpadeo de un LED mediante una señal de onda cuadrada. La capacidad de ajustar la frecuencia del parpadeo con un potenciómetro proporcionó una excelente práctica en el manejo de señales astables.

Este proyecto demuestra la utilidad del 555 en aplicaciones simples de control de tiempo y genera una base sólida para su uso en proyectos más complejos.
