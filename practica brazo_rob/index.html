Practica Brazo Robot

Mario E. Cuan Rivera y Sebastián Rodríguez Zapata
Ing. Mecatrónica
Fecha: Octubre 11, 2024
Arduino Electrónica

- Resumen -

Este proyecto del brazo robotico con servomotores consiste en el diseño y construcción de un brazo robótico utilizando Arduino como plataforma de control y servomotores para el movimiento de las articulaciones. El brazo está fabricado con MDF, lo que ofrece una estructura ligera y económica. El sistema se controla mediante potenciómetros, los cuales permiten al usuario manipular el movimiento del brazo para levantar una pequeña pelota de plástico. El objetivo principal es proporcionar una interfaz sencilla para el control manual de un brazo robótico con aplicaciones básicas en el campo de la robótica educativa y experimental.

- Introducción -

La robótica ha evolucionado como una disciplina clave en la ingeniería moderna, con aplicaciones en diversas industrias, desde la manufactura hasta la medicina. Uno de los componentes más representativos de este campo es el brazo robótico, que imita la funcionalidad de un brazo humano para realizar tareas automatizadas. En este proyecto, se busca construir un brazo robótico programado con Arduino y controlado mediante potenciómetros para levantar objetos ligeros, como una pelota pequeña de plástico. El uso de servomotores permite movimientos precisos en cada articulación del brazo, y los potenciómetros brindan un control manual intuitivo, ideal para proyectos de aprendizaje o demostraciones prácticas.

- Desarrollo -

Para poder desarrollar este proyecto de manera adecuada, primero que nada tuvimos que idear como iba a ser nuestro brazo robot; pensar en el diseño que queríamos que tuviera.

Nuestro brazo robot fue hecho completamente con corte láser (DXF), lo diseñamos en la plataforma de SolidWorks.

Después de ya tener el diseño listo y las piezas cortadas empezamos a armarlo. Para lograr que el brazo estuviera bien unido y no se cayeran las piezas o tuviera desensambles atornillamo las piezas entre sí y le pusimos tuercas para que estuviera más seguro y no se desarmara.

Una vez hecho eso empezamos con la parte electrónica. Conectamos los respectivos cables del protoboard al arduino, al igual que los servomotores. Los potenciometros, que fue con lo que pudimos hacer que el brazo se moviera, los conectamos en el protoboard.

Diseño

Para el diseño de este proyecto, como ya se menciono anteriormente, lo hicimos en la plataforma SolidWorks y fue puro corte láser. El diseño fue pensado para que pudiera agarrar una pelota sin problema alguno; un diseño chico y con buena estabilidad.

- Resultados -

El brazo robótico construido ha sido capaz de levantar con éxito la pelota y pasar la pelota por 5 brazos mas mediante el control manual de los potenciómetros. Se logró un movimiento fluido en las articulaciones gracias a la correcta programación de los servomotores en la placa Arduino. Los potenciómetros permitieron un control preciso de cada grado de libertad del brazo, facilitando la manipulación del objeto. La estructura de MDF fue lo suficientemente resistente para soportar el peso de los componentes y de la pelota sin comprometer la estabilidad del brazo.

- Conclusiones -

El proyecto del brazo robótico demostró la viabilidad de construir un sistema robótico funcional, utilizando componentes accesibles como Arduino, servomotores y MDF. El control mediante potenciómetros se reveló como una opción intuitiva para operadores sin experiencia previa en robótica, ofreciendo una interacción directa y fluida. Este proyecto tiene aplicaciones potenciales en el campo de la educación, sirviendo como base para entender principios fundamentales de robótica, electrónica y programación. Los resultados obtenidos permiten concluir que este tipo de sistemas puede escalarse o adaptarse para otros proyectos con mayores requerimientos de precisión y carga.
