Arduino Electrónica

- Resumen -

En esta práctica nuestro objetivo era que al conectar dos LEDs al Arduino los LEDs se pudieran encender de manera autónoma.

- Introducción -

Esta fue una de las primeras pruebas en las que utilizamos el Arduino.

El Arduino sirve para poder crear y desarrollar muchos proyectos y elementos autónomos. También puede conectarse a otros dispositivos o interactuar con diferentes programas para interactuar tanto con el hardware como con el software.
