Pagina Web

Mario E. Cuan Rivera
Ing. Mecatrónica
Fecha: Agosto 22, 2024
Arduino Electrónica

- Resumen -

En esta práctica nuestro objetivo era hacer una página web en donde subiéramos nuestros diferentes reportes de laboratorio y prácticas.

- Introducción -

La página web nos ayudó bastante al comprender de una mejor manera la programación y al mismo tiempo la estructuración de una página, códigos y de los reportes de las practicas

- Desarrollo -

Primero tuvimos que descargar diferentes archivos para poder realizar nuestra página y así mismo entender como funcionaba esta misma al igual que la estructuración y componentes.

Nuestros resultados fueron positivos ya que la página se realizó con éxito y con una estructura entendible.

- Conclusiones -

Como mencioné anteriormente fue un proyecto que salió como esperábamos.
